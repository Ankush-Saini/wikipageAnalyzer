**********************************************************
Wikipages Analyzer: This is solution provided for the 
screening purposes.
**********************************************************
Technologies used: Django, Django REST framework, React
**********************************************************
To Deploy Backend Code:

Requirements:Python 3.6 or above, Django,djangorestframework
libraries

You will require following steps
1. Open terminal
2. Go to directory <path_of_unzip_folder>/Python-Django/
wikipage/
3. Run command "source env/bin/activate" to setup virtual
environment
4. Go back one directory "cd .."
5. Run command "python manage.py runserver"
**********************************************************
To Deploy Frontend Code:

Requirements: Node Package Manger, React.js, Node.js
You will require following steps:
1. Project Directory for code is wikipage
2. Open terminal
3. Go to <path_of_unzip_folder>/wikipage
4. Run command "npm start"
